go to global-key-listener folder make sure you dont enter src nor native folders then run command

javac -h native src/main/java/com/example/globalkey/NativeKeyboardHook.java

then run x64 Native Tools Command Prompt for VS 2022 and then run command 

cd C:\Users\cvetk\Downloads\global-key-listener\native 

and then run command

mkdir build
cd build

WIN BUILD:
cmake .. -G "Visual Studio 17 2022"
cmake --build . --config Release

LINUX BUILD:

OPEN WSL TERMINAL FIRST
sudo apt update
sudo apt install build-essential cmake libx11-dev libboost-all-dev openjdk-21-jdk

export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64  # Adjust if needed

mkdir -p build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build .

and then inside native/build/Debug folder you should see .dll
